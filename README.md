# BigOasis_TeamDivider

**BigOasis 알고리즘 스터디 Python 트랙**을 위한 간단한 팀 나누기 프로그램입니다.

**[웹앱 바로가기](https://big-oasis-team-divider.vercel.app)**

## 기능

- 쉼표로 구분된 멤버 이름 입력
- 조 개수와 조당 인원 입력
- 무작위로 멤버를 각 조에 배정
